# OverlayApp (Streak Overlay for Dead by Daylight)
[![Join the Discord server](https://img.shields.io/discord/1456421254302142555?logo=discord&logoColor=eee&label=Discord&labelColor=464ce5&color=fff&cacheSeconds=43200)](https://discord.gg/puZuDXrcAU)

A tool for Dead by Daylight streamers to create and manage winstreak and scoreboard overlays.

## Install

**Streak Overlay for Dead by Daylight** is available for Windows at [OverlayApp.tv](https://overlayapp.tv) or via the Releases tab.

If your browser warns that the app isn't commonly downloaded, select More options (three dots), then select Keep. If Microsoft Defender SmartScreen appears, select the arrow next to Delete, then select Keep anyway.

## Support

> **Note**
> This app is in alpha and may be unstable. Save any information you enter in another location.

For help or feedback, join the [Discord server](https://discord.gg/puZuDXrcAU). You may also [open a new issue](https://github.com/superdazed/dbd-streak-overlay/issues/new).
